\# HR Analytics Dashboard — Power BI



\## Overview

An interactive HR analytics dashboard built in Power BI to analyze employee 

attrition, headcount trends, and workforce demographics for data-driven 

HR decision-making.



\## Key Questions Answered

\- What is the overall attrition rate and how does it vary by department?

\- Which employee segments (age, tenure, role) have the highest attrition?

\- How is headcount distributed across departments and job levels?



\## Tools \& Skills

\- Power BI Desktop (.pbix)

\- DAX (measures, calculated columns)

\- Data modeling (star schema)

\- Power Query (ETL/data cleaning)



\## Data Source

IBM HR Analytics Employee Attrition dataset (public dataset)



\## Key Metrics

\- Attrition Rate

\- Average Monthly Income by Role

\- Employee Satisfaction Scores

\- Headcount by Department
